hibarai Engine 機能紹介

マスターデータ管理機能

ユーザー設定、給与体系、振込口座、現場、スタッフの登録・更新をおこないます。

hibarai Engineの初期設定時、現場スタッフの新規登録・変更発生毎に使用します。
