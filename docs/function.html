機能紹介

hibarai Engineとは、日払い・週払い専用の給与計算ソフトです。日額表の給与計算を可能とし、社会保険控除、年末調整に対応します。

マスターデータ管理機能

ユーザー設定、給与体系、振込口座、現場、従業員様の登録・更新をおこないます。

hibarai Engineの初期設定時、現場スタッフの新規登録・変更発生毎に使用します。

データ入出力機能

勤怠データ・アルバイトマスターデータのアップロード、勤怠データ・アルバイトマスターデータ、給与明細データのダウンロードをおこなうことができます。

また各種履歴の表示もおこなうことができます。

給与計算機能

とりこんだ勤怠データをもとに日払い・週払いの給与計算をおこないます。

社会保険計算にも対応しております。

給与支払データ生成機能

確定した給与計算結果をもとに、銀行振込の場合は「銀行送信用のデータ（全銀データ）」を、現金支給の場合は「受領書」を生成します。

年末調整機能

年間の給与計算結果に対し、年末調整処理をおこないます。

給与計算代行サービス

お客様のご要望に合わせたフルオーダーメイドサービス

給与計算業務に関しては、法人様それぞれの給与計算の方法があり、それに応じてお客様ごとにさまざまな課題がございます。

当社の給与計算代行サービスは、hibarai Engineを使用した「日額表」の給与計算結果を代行し、振込データの作成や給与明細の発送など「日払い」をワンストップでご提供します。

多様な給与計算方法にもご対応しておりますので、ぜひ一度ご相談ください。
